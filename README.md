# TechPrakt2025API
API для технологічної практики

## Посилання на документацію
``https://documenter.getpostman.com/view/53573150/2sBXinFVPf``

## Інструкція запуску проєкту

### 1. Встановлення .NET SDK 8
Завантажити з: https://dotnet.microsoft.com/download

### 2. Клонування репозиторію
git clone https://github.com/ТВІЙ_АКАУНТ/TechPrakt2025API.git

### 3. Встановлення залежностей
dotnet restore

### 4. Запуск сервера
cd API
dotnet run

### 5. Відкрити Swagger
https://localhost:7239/swagger
